常见问题 ​

Sharkler 和 Element Plus、Antd 等有什么不同？ ​

Sharkler 是基于 StencilJS 开发的 Web Components 库，不依赖于任何前端框架，可以在任何支持 Web Components 标准技术的浏览器中使用。

Vue 模板中属性绑定无效？ ​

对于 Object 和 Array ，请使用 .prop 修饰符，详见 Vue 官方仓库 issues#2343

vue
<template>
	<sk-checkbox-group
		:value.prop="['男', '2']"
		:options.prop="[
			'男',
			{
				value: '1',
				label: '女'
			},
			{
				value: '2',
				label: '武装直升机'
			},
			{
				value: '3',
				label: '沃尔玛塑料袋',
				disabled: true
			}
		]"
	/>
</template>

驼峰式属性，使用短横线命令或者 .prop 修饰符

vue
<template>
	<sk-label>
		{{ value }}
		<sk-switch
			:leftValue.prop="'哈哈'"
			:rightValue.prop="'呵呵'"
			:checked="checked"
			@sk-change="handleChange"
		/>
	</sk-label>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const checked = ref(false)
const value = ref('呵呵')

const handleChange = (
	ev: CustomEvent<{
		checked: boolean
		value: any
	}>
) => {
	checked.value = ev.detail.checked
	value.value = ev.detail.value
}
</script>
呵呵

为什么事件名以 sk- 开头 ​

因为 Sharkler 组件派发的是 CustomEvent，加前缀是为了避免与原生 DOM 事件冲突

为什么是 Props？ ​

Props 是暴漏在 HTML 元素上的响应式的 attributes/properties，既是 DOM 的 property，同时也存在于 DOM 的 attributes 中，所以可以像下面这样使用

vue
<template>
	<sk-button @click="clickToLoading">Click me</sk-button>
</template>

<script setup lang="ts">
const clickToLoading = (ev) => {
	const btn = ev.target

	// set property
	btn.loading = true
	// or attribute
	// btn.setAttribute('loading', 'true')

	setTimeout(() => {
		// set property
		btn.loading = false
		// or attribute
		// btn.setAttribute('loading', 'false')
	}, 2000)
}
</script>
