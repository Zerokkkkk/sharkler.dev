兼容性 ​

Sharkler 仅支持现代浏览器，如果确需支持旧版本的浏览器，请自行添加 Babel 和相应的 Polyfill。

Edge ≥ 84Chrome ≥ 84Firefox ≥ 84Safari ≥ 12

浏览器引入 ​

使用 HTML 标签引入的方式，Sharkler 会在 window 上挂载 Shark 全局对象

html
<head>
	<link rel="stylesheet" href="http://25.75.7.56:30001/cdn/sharkler@latest/sharkler.css" />
	<script type="module" src="http://25.75.7.56:30001/cdn/sharkler@latest/sharkler.esm.js"></script>
</head>

<body>
	<sk-button onclick="Shark.message('Hello World')">show message</sk-button>
	<sk-button onclick="Shark.notify('Hello World')">show notify</sk-button>
</body>
